# node-typescript

This system allows you to save your tasks

## fields   
- title
- description
- duration
- author


# To fetch all tasks endpoint
http://localhost:3000/tasks

# To fetch all tasks based on author endpoint
http://localhost:3000/tasks/:author

http://localhost:3000/tasks/alice
